# SCT_ML_3
Project Overview
This project performs image classification to distinguish between dogs and cats using a machine learning/deep learning model.
The model is trained on a labeled dataset of dog and cat images.
The dataset used in this project is taken from Kaggle:
Dataset Name: Dog vs Cat
Source: Kaggle
 https://www.kaggle.com/datasets/anthonytherrien/dog-vs-cat
